- [Java 입문 수업 (생활코딩)](https://opentutorials.org/module/516/4551)이 체계적으로 정리가 잘 되어있다고 들었다. 그래서 이 사이트 및 링크된 유튜브 영상을 통해 Java에 입문한다

- 실습 내용을 따라하는 것 뿐만 아니라 이해를 해야한다
